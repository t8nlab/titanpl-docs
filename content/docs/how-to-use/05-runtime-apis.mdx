## Overview

Titan Planet provides a comprehensive set of **runtime APIs** exposed via the global `t` object or through the `@titanpl/native` utility package. These APIs execute directly in the high-performance Rust core, offering near-native speed for system-level operations.

### Installation & Import

While `t` is always available globally, you can also import specific utilities for better type support and clarity:

```javascript
import { fs, ws, crypto, os, response } from "@titanpl/native";

export const myAction = (req) => {
  const content = fs.readFile("config.json"); // Uses project root as base
  return response.json({ content });
};
```

## API Reference

### `fs` (File System)
Synchronous file system operations.

- `fs.readFile(path: string): string` ‚Äî Read file content as UTF-8 string.
- `fs.writeFile(path: string, content: string): void` ‚Äî Write string content to file.
- `fs.exists(path: string): boolean` ‚Äî Check if a path exists.
- `fs.mkdir(path: string): void` ‚Äî Create a directory (recursive).
- `fs.remove(path: string): void` ‚Äî Remove file or directory.
- `fs.readdir(path: string): string[]` ‚Äî List directory contents.
- `fs.stat(path: string): object` ‚Äî Get file statistics `{ isFile: boolean, isDir: boolean, size: number, modified: number }`.

### `path` (Path Manipulation)
Utilities for handling file paths.

- `path.join(...parts: string[]): string` ‚Äî Join path segments using platform-specific separators.
- `path.resolve(...parts: string[]): string` ‚Äî Resolve path to an absolute path.
- `path.dirname(path: string): string` ‚Äî Get the directory name of a path.
- `path.basename(path: string): string` ‚Äî Get the base name (filename) of a path.
- `path.extname(path: string): string` ‚Äî Get the extension of a path (including the dot).

### `crypto` (Cryptography)
Secure cryptographic utilities.

- `crypto.hash(algo: string, data: string): string` ‚Äî Hash data (e.g., `sha256`, `sha512`).
- `crypto.randomBytes(size: number): string` ‚Äî Generate random bytes as a hex string.
- `crypto.uuid(): string` ‚Äî Generate a UUID v4.
- `crypto.encrypt(algorithm: string, key: string, plaintext: string): string` ‚Äî Encrypt text using native Rust implementations.
- `crypto.decrypt(algorithm: string, key: string, ciphertext: string): string` ‚Äî Decrypt text.
- `crypto.hashKeyed(algo: string, key: string, message: string): string` ‚Äî Keyed-hash (HMAC) support.
- `crypto.compare(hash: string, target: string): boolean` ‚Äî Securely compare two strings in constant time.

### `buffer` (Buffer Utilities)
Utilities for binary and data encoding.

- `buffer.fromBase64(str: string): Uint8Array` ‚Äî Decode Base64 string to bytes.
- `buffer.toBase64(bytes: Uint8Array | string): string` ‚Äî Encode bytes or string to Base64.

### `os` (Operating System)
Retrieve system-level information.

- `os.platform(): string` ‚Äî OS platform (e.g., `linux`, `windows`, `darwin`).
- `os.cpus(): number` ‚Äî Number of logical CPU cores.
- `os.totalMemory(): number` ‚Äî Total system memory in bytes.
- `os.freeMemory(): number` ‚Äî Free system memory in bytes.
- `os.tmpdir(): string` ‚Äî Path to the system temporary directory.

### `net` (Network)
Basic network utilities.

- `net.resolveDNS(hostname: string): string[]` ‚Äî Resolve a hostname to IP addresses.
- `net.ip(): string` ‚Äî Get the local system IP address.

### `ws` (WebSockets)
Real-time messaging utilities for TitanPl WebSocket actions.

- `ws.send(socketId: string, message: string): void` ‚Äî Send a message to a specific WebSocket connection.
- `ws.broadcast(message: string): void` ‚Äî Broadcast a message to all connected clients.

### `proc` (Process Management)
Manage and query system processes.

- `proc.pid(): number` ‚Äî Get the Process ID of the current Titan runtime.
- `proc.info(): object` ‚Äî Get basic info about the current process.
- `proc.run(command: string, args?: string[], cwd?: string): object` ‚Äî Spawn a background process.
- `proc.kill(pid: number): boolean` ‚Äî Terminate a process by PID.
- `proc.list(): object[]` ‚Äî List all running system processes with CPU and memory usage.

### `time` (Time Utilities)
- `time.sleep(ms: number): void` ‚Äî Synchronously block execution for `ms` milliseconds.
- `time.now(): number` ‚Äî Returns `Date.now()`.

### `ls` (Persistent Local Storage)
High-performance key-value storage persisted to disk.

- `ls.get(key: string): string|null` ‚Äî Retrieve a string value.
- `ls.set(key: string, value: string): void` ‚Äî Persist a string value.
- `ls.remove(key: string): void` ‚Äî Delete a key.
- `ls.clear(): void` ‚Äî Clear all stored data.
- `ls.keys(): string[]` ‚Äî List all stored keys.
- `ls.setObject(key: string, value: any): void` ‚Äî Store a complex JS object using native V8 serialization.
- `ls.getObject(key: string): any` ‚Äî Retrieve and restore a complex JS object.
- `ls.serialize(value: any): Uint8Array` ‚Äî Native V8 serialization (supports Map, Set, Date, Uint8Array).
- `ls.deserialize(bytes: Uint8Array): any` ‚Äî Native V8 deserialization.

### `session` (Session Management)
- `session.get(sid: string, key: string): string|null` ‚Äî Get session data.
- `session.set(sid: string, key: string, value: string): void` ‚Äî Set session data.
- `session.delete(sid: string, key: string): void` ‚Äî Delete session data.
- `session.clear(sid: string): void` ‚Äî Clear an entire session.

### `cookies` (HTTP Cookies)
- `cookies.get(req: object, name: string): string|null` ‚Äî Extract cookie value from request.
- `cookies.set(res: object, name: string, value: string, options?: object): void` ‚Äî Attach `Set-Cookie` to response.

### `url` (URL Parsing)
- `url.parse(str: string): URL` ‚Äî Parse a URL string into a native URL object.
- `new url.SearchParams(init?: string)` ‚Äî Construct query parameters.

### `response` (HTTP Response Builder)
Helper for constructing standardized Titan responses.

- `response.text(content: string, options?: object): object` ‚Äî Create a plain text response.
- `response.json(data: any, options?: object): object` ‚Äî Create a JSON response.
- `response.html(content: string, options?: object): object` ‚Äî Create an HTML response.
- `response.redirect(url: string, status?: number): object` ‚Äî Create a redirect response.
