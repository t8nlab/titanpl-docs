## ðŸš€ Create first server
Create your fist titan server using ```tit init```

```bash
tit init my-app
cd my-app
tit dev
```

Titan will:

* Build routes
* Bundle actions
* Start Rust dev server
* Watch file changes
* Trigger instant reload
